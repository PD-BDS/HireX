# HireX – Quick‑Start User Guide

## What is HireX?
HireX is an AI‑powered recruitment assistant chat application that lets you build sturctured **Job description**, **Screen Candidates** and **Discuss Results** directly from a web interface.

## User Instructions
Follwoing the below insturction will help you to test the application effectively, without running into any issues.
1. **Open the app** – Navigate to the URL where the frontend is hosted (`https://hirex-zp9p.onrender.com`).
2. **Wait for app to load** – Wait for 5-10 minutes for the app to load with all the data. If you start conversation before the application server is ready, you will not get any response, and get error. Therefore, Requesting your patience. After Successful loading of the app, you will see a Existing session chats present on the left side, and the chats sessions are loading in the app when clicked.
3. **Application Usage** – Do not run multiple things in the application at the same time. Wait for one command to finish processing and update the app before running another command.
4. **Have Patience** – The application performs several background calls (LLM completions, vector‑search, Storage calls). **Typical response time is 5-10 minutes**. A spinner (`Thinking…`) indicates the system is still processing. **Do not start any other processes meanwhile**; the application will finish the current task before handling the next.

## Getting Started
1. **Create a New Chat** –  Click the **New Chat** button to start a new conversation.
2. **Enter your job description** – send message to the assistant with your job requirements for a role. it will guide you to build a structured job description. It does not generate a job description automatically, it extracts what you provide as the job description and make it structured, for your review and Running search for candidates. you can see the updated job description in the **Job Snapshot** inside the **Active Context** tab.
3. **Start the screening** – When you are satisfied with the extracted **Job Snapshot** you can write to the assitant to start screening for candidates. and it will start running search for candidates and it will show you the results in the chat, with their matches, and reasoning behind their selection.
4. **Discuss the results** – you can ask the assistant to provide more details about the candidates, or comparison between candidates from the results, to assist you in making a decision.


Enjoy using HireX to streamline your hiring workflow!
